# Tutorials

Step-by-step guides for getting the most out of Razer Lighting. Start from the top if you're new, or jump to a specific tutorial.

| # | Tutorial | What you'll learn | Time |
|---|---|---|---|
| 1 | [Running Your First Effect](01-your-first-effect.md) | Launch the app, pick an effect, and see your keyboard light up | 5 min |
| 2 | [Using the Config GUI](02-using-the-config-gui.md) | Open the configuration window, tweak parameters, preview and save changes | 10 min |
| 3 | [Customizing Colors](03-customizing-colors.md) | Edit palettes and individual colors in the GUI and by hand in config files | 10 min |
| 4 | [Writing a Custom Effect](04-writing-a-custom-effect.md) | Create a new effect from scratch with config file and auto-discovery | 20 min |
| 5 | [Using the Shared Utilities](05-using-shared-utilities.md) | Use palette LUTs, frame timing, grid math, and numpy rendering | 15 min |
| 6 | [Capturing GIF Previews](06-capturing-gifs.md) | Record any effect as an animated GIF with config overrides | 5 min |
| 7 | [Autostart and Standalone Mode](07-autostart-and-standalone.md) | Run effects at login, run standalone without the tray app | 5 min |

## Prerequisites

All tutorials assume you have completed the [Getting Started](../getting-started.md) guide — Razer Lighting is installed, OpenRazer is running, and your keyboard responds to the tray app.

Tutorials 4 and 5 involve writing Python code. Basic Python knowledge (variables, loops, functions) is required.
